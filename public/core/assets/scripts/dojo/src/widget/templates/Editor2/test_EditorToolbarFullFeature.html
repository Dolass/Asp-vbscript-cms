Thanks to Bruce Webster from Interact Learning Community Environment for contributing this full featured Editor2 toolbar template and corresponding stylesheets/images. This is a sample page to show how to use this new template file. In addition, this page also deploys several Editor2 plugins:

  • ContextMenu: right click in the editor to see it
  • FindReplace: can be access by find/replace toolbar buttons
  • TableOperation: insert/delete table

In this example, default toolbarTemplatePath and toolbarTemplateCssPath in Editor2 are overwritten.
